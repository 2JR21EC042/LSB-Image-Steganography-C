# LSB Image Steganography in C

This project implements image steganography using the Least Significant Bit (LSB) technique in C.

## Features
- Encode secret message inside BMP image
- Decode hidden message
- Bitwise manipulation
- File handling in C

## Technologies
- C Programming
- Bitwise operations
- BMP image processing

## How it Works
1. Read BMP image
2. Convert secret message to binary
3. Embed bits into image pixels
4. Generate stego image
5. Decode message from stego image
